# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by opening a GitHub issue or contact the maintainers. All security vulnerabilities will be promptly addressed.

## Disclosure Policy

When a vulnerability is reported, we will:
1. Acknowledge the receipt of the report within 48 hours.
2. Provide an estimated timeframe for a fix.
3. Notify the reporter once the vulnerability is fixed.
4. Publicly disclose the vulnerability after a fix has been released and enough time has passed for users to update.
