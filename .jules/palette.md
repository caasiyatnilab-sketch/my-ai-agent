## 2025-05-15 - CLI Visual Feedback and Scannability
**Learning:** Users perceive long-running CLI operations (like LLM calls) as 'hung' without immediate visual feedback. A transient 'Thinking...' indicator improves perceived responsiveness. Additionally, separating machine-like tool outputs from human-like assistant responses with whitespace significantly improves readability.
**Action:** Always include a visual indicator for long-running CLI tasks and use whitespace to delimit different sections of the output.
